# ShoppingMiao
购物喵秒杀、抢购、低价购等内容的研究、测试。请不要用于京东、淘宝、天猫、小米有品、苏宁易购、考拉海购、唯品会等平台的秒杀、抢购、低价购等。<br>
请注意：本插件仅可用于研究测试使用，不可用于其他用途。使用于其他用途，后果自负。<br>
